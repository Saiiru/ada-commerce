# Contribuição

## Fluxo
1) Abra uma **Issue** com escopo e critérios de aceite.
2) Crie branch `feat/<escopo>` ou `fix/<escopo>`.
3) Commits em **Conventional Commits**.
4) Abra PR pequeno. 1 review. Merge via **Squash**.

## Padrões
- Java 17. Clean Architecture.
- Testes JUnit 5.
- Sem REST na Fase 1.
